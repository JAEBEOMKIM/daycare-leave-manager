## 기술 스택
- **Frontend**: React 18 + Next.js 16
- **Backend**: Next.js API Routes + Supabase
- **Database**: PostgreSQL (Supabase)
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Deployment**: Vercel
- **Version Control**: GitHub
