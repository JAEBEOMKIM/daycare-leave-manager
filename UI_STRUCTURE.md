# 화면 구조 설계

## 전체 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  Header: 어린이집명 | 현재날짜 | 사용자                  │
├──────────────┬──────────────────────────────────────────┤
│              │                                          │
│  Sidebar     │         Main Content Area               │
│              │                                          │
│ - 대시보드    │                                          │
│ - 통계        │                                          │
│ - 직원관리    │                                          │
│ - 개인별조회  │                                          │
│ - 연차등록    │                                          │
│ - 기준설정    │                                          │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

## 색상 스킴
- Primary: #0058c3 (Professional Blue)
- Success: #067647 (Green)
- Warning: #f79009 (Orange)
- Error: #ba1a1a (Red)
- Background: #f9f9ff
- Border: #c2c6d7

## 반응형 디자인
- Desktop: Full layout
- Tablet: 2-column layout (Sidebar + Content)
- Mobile: Stacked layout (Sidebar collapses to hamburger menu)
